Security policy

If you discover a vulnerability in this Home Assistant integration, please open an issue and mark it private or contact the maintainer directly. Do not publish vulnerability details until a fix is available.

Credentials
- Do not commit Home Assistant secrets or plaintext credentials to this repo.
- The integration uses Home Assistant config entries to store tokens; users should use the UI config flow. For manual setups, use Home Assistant's secrets.yaml.

Reporting
- Include steps to reproduce and affected versions.
